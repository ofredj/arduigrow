ArduiGrow
=========

A monitoring and automatic growing system based on PcDuino/linduino and arduino probes.

Features
--------

- PH meter
- EC meter
- Temperature
- Humidity
- Water Tank Level
- Pumps (empty/fill water tank)
- Webcam
- HTTP interface
- Email Alerts

Compiling ino sketch
--------------------

cd ino && make all
